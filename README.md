# Emergency Department Analysis
Modelled and developed a 3-NF normalised SQL database looking at presentations and presentation times in Emergency Department across Australia sorted by triage categories, states, episode end status, years and funding sources.
Developed a dashboard about Emergency Departments in Victoria in 2024-25 looking to provide actionable insights to directors in emergency department across Victoria 

# Tools used
+ SQLlite
+ Python for ETL
+ PowerBI for dashboard
+ Lucid chart for ERD schema

# Dashboards
The latest dashboard dashboard_based_on_duration help ED directors in Victoria to make decisons regarding ICU/HDU bed availability, investigation turnaround times and specialist review delays
